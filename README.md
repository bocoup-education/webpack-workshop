# webpack-workshop
> Learn Webpack from the basics to the advanced

## Setup Instructions

- install [Node.js](https://nodejs.org)
- npm install
- npm install -g webpack

### Concepts Covered
- Basic Core Use with HTML, JS, and CSS
- Handle Incompatible Dependencies
- Debugging / Testing
- Convenience Modules
- Hot Module Replacement (HMR)
- Code Splitting
